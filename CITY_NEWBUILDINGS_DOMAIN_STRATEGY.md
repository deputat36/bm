# Домен и стратегия городского сайта по новостройкам Борисоглебска

## Выбранный основной домен

`novostroyki-borisoglebsk.ru`

Статус: выбран как основной домен будущего городского сайта по новостройкам Борисоглебска.

Почему выбран именно он:

- понятен людям;
- содержит главный поисковый запрос;
- не имитирует официальный сайт одного застройщика;
- подходит под каталог разных проектов;
- позволяет разместить ЖК «Теллерманов сад» как один из разделов;
- подходит для рекламы не одного объекта, а услуги подбора новостроек.

## Рабочее позиционирование

Новостройки Борисоглебска — городской каталог новых домов, квартир, планировок и ипотечных программ.

Ключевая идея: не официальный сайт конкретного ЖК, а независимый информационный каталог и консультационный сервис для покупателей новостроек в Борисоглебске.

## Роль ЖК «Теллерманов сад»

ЖК «Теллерманов сад» размещается внутри каталога как отдельная карточка/раздел:

`/catalog/tellermanov-sad/`

В разделе можно разместить:

- обзор проекта;
- квартиры;
- планировки;
- документы;
- FAQ;
- галерею;
- форму консультации.

## Не рекомендовать как основной домен

- домены с названием конкретного ЖК;
- домены с названием BM Group;
- домены со словом official / официальный;
- домены, создающие впечатление сайта застройщика;
- слишком брендовые домены, которые нельзя масштабировать на разные объекты.

## Структура будущего сайта

- `/` — новостройки Борисоглебска, общий вход;
- `/catalog/` — каталог проектов;
- `/catalog/tellermanov-sad/` — раздел ЖК «Теллерманов сад»;
- `/catalog/tellermanov-sad/kvartiry/` — квартиры;
- `/catalog/tellermanov-sad/planirovki/` — планировки;
- `/catalog/tellermanov-sad/dokumenty/` — документы;
- `/ipoteka/` — ипотека и господдержка;
- `/guides/` — инструкции покупателям;
- `/news/` — новости рынка;
- `/contacts/` — консультация.

## Роль текущего сайта tellermanovsad.ru

До согласования с BM Group:

- не использовать как основной рекламный домен;
- не запускать широкий платный трафик;
- использовать как временный рабочий прототип;
- постепенно переносить материалы в структуру общего сайта.

После запуска нового сайта:

1. Перенести контент в раздел `/catalog/tellermanov-sad/`.
2. Закрыть старый домен от индексации или поставить редирект.
3. Оставить старый домен только при письменном согласовании.

## Как выстраивать работу

### Этап 1. Подготовка

- купить домен `novostroyki-borisoglebsk.ru`;
- подключить домен к хостингу или GitHub Pages;
- создать новый репозиторий или отдельную ветку;
- утвердить структуру сайта;
- собрать список всех новостроек Борисоглебска;
- подготовить нейтральное позиционирование.

### Этап 2. MVP

- главная страница;
- каталог новостроек;
- карточка ЖК «Теллерманов сад»;
- базовые страницы по ипотеке;
- форма консультации;
- техническая SEO-основа.

### Этап 3. Контент

- обзор каждой новостройки;
- статьи для покупателей;
- FAQ;
- ипотечные инструкции;
- сравнение районов и форматов квартир.

### Этап 4. Реклама

- сначала органика, городские группы, нативные посты;
- потом тестовая реклама на общий каталог;
- не использовать брендовые запросы конкретных застройщиков без согласования;
- не использовать формулировки, запрещенные регламентами застройщиков.

### Этап 5. CRM и аналитика

- все формы должны собирать имя, телефон, интерес, бюджет/способ покупки;
- заявки отправлять в почту и затем в Supabase/CRM;
- размечать источники UTM;
- отдельно фиксировать объект интереса.

## Рекламное позиционирование

Безопасные формулировки:

- Новостройки Борисоглебска в одном месте.
- Поможем сравнить новые дома и подобрать квартиру.
- Каталог новых домов Борисоглебска.
- Консультация по покупке квартиры в новостройке.
- Ипотека и подбор квартиры в новом доме.

Не использовать без согласования:

- официальный сайт ЖК;
- квартиры от застройщика;
- дешевле, чем у застройщика;
- напрямую от застройщика;
- официальный партнер BM Group;
- официальный представитель проекта.
