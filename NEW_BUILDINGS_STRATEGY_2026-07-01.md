# Новая стратегия: сайт по всем новостройкам Борисоглебска

Дата: 2026-07-01

## Причина смены стратегии

По регламенту BM Group отдельный сайт объекта с названием ЖК в домене несет договорной риск. Более безопасная стратегия — создать общий информационный сайт по новостройкам Борисоглебска, а ЖК «Теллерманов сад» разместить внутри него как отдельный раздел.

## Целевая структура

Главный сайт:

- новостройки Борисоглебска
- каталог жилых комплексов
- ипотека и господдержка
- новости рынка
- полезные материалы покупателю
- контакты специалиста

Раздел ЖК «Теллерманов сад»:

- обзор проекта
- квартиры
- планировки
- цены после подтверждения
- документы
- ход строительства
- FAQ
- форма квалифицированной заявки

## Что перенести из текущего сайта

- официальные опубликованные данные по объекту
- квартирографию
- страницы документов
- FAQ
- галерею после проверки материалов
- форму заявки с квалификацией
- предупреждение, что заявка не является бронью

## Что убрать или не использовать

- домен с названием одного ЖК в публичной рекламе без согласования
- формулировки вида квартиры от застройщика
- имитацию официального сайта
- отдельные соцсети объекта от имени проекта
- оффлайн-рекламу без согласования

## Безопасная рекламная стратегия

До согласования:

- не запускать широкий платный трафик на tellermanovsad.ru
- использовать сайт как внутренний рабочий инструмент
- вести клиентов через личные консультации и квалифицированные заявки

После запуска общего сайта:

- продвигать общий сайт по запросам новостройки Борисоглебска
- делать разделы по каждому проекту
- не использовать брендовые запросы BM Group и названия ЖК в контекстной рекламе без согласования
- готовить публикации как обзор рынка новостроек, а не как официальный сайт конкретного ЖК

## Ближайшие технические шаги

1. Подготовить структуру нового сайта по всем новостройкам.
2. Создать карту разделов и URL.
3. Перенести текущие материалы в раздел нового сайта.
4. Закрыть или законсервировать tellermanovsad.ru.
5. Подготовить пакет материалов для согласования BM Group.
