# Карта будущего сайта по новостройкам Борисоглебска

## Главная идея

Создать не отдельный сайт одного ЖК, а городской информационный ресурс по всем новостройкам Борисоглебска. Раздел ЖК «Теллерманов сад» должен быть частью каталога.

## Возможная структура URL

- `/` — главная: новостройки Борисоглебска
- `/catalog/` — каталог новостроек
- `/catalog/tellermanov-sad/` — раздел ЖК «Теллерманов сад»
- `/catalog/tellermanov-sad/kvartiry/` — квартиры
- `/catalog/tellermanov-sad/planirovki/` — планировки
- `/catalog/tellermanov-sad/ceny/` — цены после подтверждения
- `/catalog/tellermanov-sad/dokumenty/` — документы
- `/catalog/tellermanov-sad/faq/` — вопросы
- `/ipoteka/` — ипотека и господдержка
- `/news/` — новости рынка новостроек
- `/guides/` — инструкции покупателю
- `/contacts/` — консультация специалиста

## Что переносим из текущего сайта

- данные объекта из опубликованных источников
- квартирографию
- страницы документов
- FAQ
- общие статьи про покупку новостройки
- формы квалифицированных заявок
- заметки по ипотеке и подготовке к покупке

## Что закрываем или не переносим

- страницы с рискованными формулировками
- страницы, имитирующие официальный сайт одного объекта
- старые рекламные посадочные без согласования
- страницы под брендовые запросы застройщика

## SEO-логика

Продвигать общий сайт по нейтральным запросам:

- новостройки Борисоглебск
- квартиры в новостройках Борисоглебска
- новые дома Борисоглебск
- ипотека на новостройку Борисоглебск
- купить квартиру в новом доме Борисоглебск

Не использовать в контекстной рекламе без согласования:

- название конкретного ЖК
- название BM Group
- название застройщика
- точный адрес проекта

## Рекламная логика

Для рекламы использовать общий вход:

- каталог новостроек
- помощь с выбором новостройки
- консультация по ипотеке
- подбор квартиры в новом доме

Конкретный ЖК показывать как один из вариантов внутри консультации или каталога.
