# Чек-лист запуска novostroyki-borisoglebsk.ru

## 1. Купить домен

Основной домен:

`novostroyki-borisoglebsk.ru`

## 2. Решить, где размещать сайт

Варианты:

1. GitHub Pages — быстро для статического сайта.
2. Beget/хостинг — удобно для PHP и будущей CRM.
3. VPS — если будет полноценный backend.

Для MVP достаточно GitHub Pages или обычного хостинга.

## 4. Репозиторий

Варианты:

1. Создать новый репозиторий под городской сайт.
2. Использовать текущий репозиторий как базу и потом переименовать.
3. Сделать отдельную ветку `city-newbuildings`.

Рекомендация: новый репозиторий, чтобы не смешивать временный сайт ЖК и городской каталог.

## 7. SEO

Создать:

- `robots.txt`
- `sitemap.xml`
- `404.html`
- `privacy/`

Открыть для индексации только публичные страницы.
Закрыть служебные формы, UTM и внутренние страницы.

## 8. Формы

На старте:

- Web3Forms;
- обязательный комментарий;
- UTM;
- согласие на обработку данных.

Следующий этап:

- Supabase;
- CRM;
- статусы заявок.

## 9. Что сделать со старым сайтом

До запуска нового сайта:

- оставить как временный прототип;
- не запускать широкий трафик;
- не продвигать как отдельный сайт ЖК.

После запуска нового сайта:

- перенести материалы;
- закрыть старый сайт от индексации или поставить редирект;
- оставить только при письменном согласовании.

## 10. Реклама после запуска

Безопасная реклама:

- общий каталог новостроек;
- подбор квартиры в новом доме;
- ипотечная консультация;
- сравнение новостроек.

Не использовать без согласования:

- брендовые запросы ЖК;
- брендовые запросы застройщиков;
- формулировки про официальный сайт;
- формулировки про прямую продажу от застройщика.
