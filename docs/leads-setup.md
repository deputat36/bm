# Настройка приёма заявок с сайта tellermanovsad.ru

## Текущее состояние

Формы на сайте уже собирают:

- имя;
- телефон;
- интересующий тип квартиры или консультации;
- комментарий;
- страницу отправки;
- заголовок страницы;
- источник перехода;
- UTM-метки;
- realtor и realtor_id;
- дату отправки.

Код находится в файле `assets/js/main.js`.

## Рекомендуемый быстрый вариант: Web3Forms

Для статического сайта на GitHub Pages самый быстрый вариант — Web3Forms. Он не требует PHP, сервера и базы данных.

## Шаг 1. Получить access key

1. Открыть сайт Web3Forms.
2. Указать email, на который должны приходить заявки.
3. Подтвердить email.
4. Получить access key.

## Шаг 2. Вставить ключ в код

Открыть файл:

`assets/js/main.js`

Найти блок:

`WEB3FORMS_ACCESS_KEY: "",`

Вставить ключ между кавычками.

Пример:

`WEB3FORMS_ACCESS_KEY: "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx",`

После этого формы начнут отправлять заявки через Web3Forms.

## Шаг 3. Проверить отправку

Проверить формы на страницах:

- главная;
- `/contacts/`;
- `/kvartiry/`;
- `/ceny/`;
- `/ipoteka/`.

Тестовая заявка должна прийти на email, который указан в Web3Forms.

## Что должно быть в письме

В письме должны быть:

- проект: ЖК Теллерманов сад;
- имя клиента;
- телефон;
- интерес;
- комментарий;
- страница отправки;
- UTM-метки;
- дата заявки.

## Альтернативный вариант: свой endpoint

Если заявки нужно отправлять в CRM, Supabase или собственный обработчик, можно использовать поле:

`LEAD_ENDPOINT: "",`

В него нужно вставить URL обработчика.

Если заполнен `WEB3FORMS_ACCESS_KEY`, заявки отправляются через Web3Forms.

Если Web3Forms не заполнен, но заполнен `LEAD_ENDPOINT`, заявки отправляются на свой endpoint.

Если оба поля пустые, форма работает в тестовом режиме и не отправляет заявку наружу.

## Что обязательно сделать до рекламы

1. Добавить страницу политики обработки персональных данных.
2. Добавить чекбокс согласия в формы.
3. Подключить Яндекс.Метрику.
4. Настроить цели: отправка формы, клик по телефону, переходы в квартиры, цены, ипотеку, документы.
5. Отправить тестовые заявки с телефона и компьютера.
6. Проверить, что заявка приходит быстро и не попадает в спам.

## Рекомендация

Для первого запуска рекламы лучше использовать Web3Forms. Это быстрее и надёжнее, чем сразу писать собственный обработчик. После проверки спроса можно подключить CRM или Supabase.
