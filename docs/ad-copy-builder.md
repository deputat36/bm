# Генератор рекламных текстов

## Назначение

Инструмент `/tools/ad-copy-builder/` помогает быстро собрать текст для VK-поста, короткого объявления или сторис по дому на Просторной 4А.

Он нужен, чтобы риэлторы и маркетинг использовали единые безопасные формулировки:

- без обещаний брони;
- без неподтверждённых цен;
- без обещаний лучших квартир;
- с понятной посадочной страницей;
- с акцентом на заявку или консультацию.

## Адрес

```text
https://deputat36.github.io/bm/tools/ad-copy-builder/
```

Страница закрыта от индексации через `noindex, nofollow`.

## Сценарии

- подбор квартиры;
- консультация по новостройке;
- ранний доступ к информации;
- ипотека;
- документы.

## Как использовать

1. Выбрать сценарий.
2. Выбрать формат: пост, объявление или сторис.
3. Добавить имя риэлтора или подпись, если нужно.
4. Скопировать текст.
5. Собрать ссылку через `/tools/utm-builder/`.
6. Проверить, что посадочная страница соответствует объявлению.
