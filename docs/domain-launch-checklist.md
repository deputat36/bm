# Чек-лист подключения домена к GitHub Pages

Основной рекомендуемый домен: `tellermanovsad.ru`

## 1. Купить домен

Проверить и купить, если свободен:

- `tellermanovsad.ru` — основной;
- `zhk-tellermanov-sad.ru` — SEO-редирект;
- `теллермановсад.рф` — офлайн/наружная реклама;
- `zhkprostornaya4a.ru` — адресный SEO-редирект.

## 2. Настроить DNS

Для основного домена в DNS добавить A-записи GitHub Pages:

```text
185.199.108.153
185.199.109.153
185.199.110.153
185.199.111.153
```

Для `www` добавить CNAME:

```text
www -> deputat36.github.io
```

## 3. Настроить GitHub Pages

В репозитории `deputat36/bm`:

1. Settings → Pages.
2. Custom domain: `tellermanovsad.ru`.
3. Дождаться проверки DNS.
4. Включить `Enforce HTTPS`.

## 4. Добавить CNAME в репозиторий

Создать файл `CNAME` в корне репозитория:

```text
tellermanovsad.ru
```

Не добавлять CNAME заранее, пока домен не куплен и DNS не настроены.

## 5. Обновить абсолютные URL

После подключения домена заменить в `sitemap.xml` и мета-данных:

- `https://deputat36.github.io/bm/`
- на `https://tellermanovsad.ru/`

Проверить:

- `sitemap.xml`;
- `robots.txt`;
- `og:image`;
- structured data JSON-LD;
- ссылки в рекламных шаблонах.

## 6. Подключить заявки

До запуска рекламы подключить обработчик форм:

- Web3Forms;
- или Supabase Edge Function;
- или другой CRM-обработчик.

Проверить заявки с:

- главной;
- `/contacts/`;
- `/kvartiry/`;
- `/ceny/`;
- `/ipoteka/`.

## 7. Аналитика

Подключить Яндекс.Метрику и цели:

- клик по телефону;
- отправка формы;
- переход в галерею;
- переход на страницу документов;
- переход на страницу квартир.

## 8. Проверка перед рекламой

- открыть сайт с телефона;
- проверить скорость загрузки;
- проверить галерею;
- проверить телефон `8 903 857-69-09`;
- проверить, что цена не обещается, если она ещё не опубликована;
- проверить, что заявка не называется бронью;
- проверить страницу документов;
- проверить `sitemap.xml`.
