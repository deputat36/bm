# Скрипт первого звонка

## Назначение

Инструмент `/tools/call-script/` нужен для обработки заявок с сайта Просторной 4А.

Он помогает:

- не забыть ключевые вопросы;
- одинаково объяснять статус предварительной заявки;
- определить приоритет лида;
- сформировать заметку для CRM или таблицы;
- назначить следующий шаг.

## Адрес

```text
https://deputat36.github.io/bm/tools/call-script/
```

Страница закрыта от индексации через `noindex, nofollow`.

## Основной принцип звонка

Не продавать квартиру, которой ещё нет в открытой продаже. Задача первого звонка — уточнить потребность и подготовить клиента к получению подтверждённых данных.

## Формулировка

```text
Здравствуйте, меня зовут ____. Вы оставляли заявку по новому дому на Просторной 4А. Уточню пару вопросов, чтобы после появления подтверждённых данных направить вам только подходящую информацию.
```

## Что обязательно уточнить

1. Комнатность.
2. Цель покупки.
3. Способ покупки.
4. Бюджет.
5. Срок покупки.
6. Ключевые критерии.
7. Следующий шаг.

## Что нельзя обещать

- бронь квартиры;
- фиксацию цены;
- скидку;
- конкретную площадь;
- конкретную планировку;
- одобрение ипотеки;
- официальный статус сайта.

## Приоритеты

- A — готов обсуждать покупку, есть бюджет или ипотека.
- B — интерес есть, но не хватает данных.
- C — ранний прогрев, хочет следить за проектом.
- D — неактуальный или ошибочный контакт.
