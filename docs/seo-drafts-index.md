# Индекс SEO-черновиков

Дата фиксации: 2026-07-05

## 1. Назначение

Файл:

```text
/data/pages/seo-drafts.json
```

хранит отдельный список SEO-страниц, которые уже созданы в структуре сайта, но пока не готовы к индексации.

Это нужно, чтобы:

- не перегружать основной индекс страниц;
- отдельно отслеживать коммерческие и справочные посадочные страницы;
- не включать черновики в sitemap раньше времени;
- видеть, какие страницы требуют проверки текста, данных и юридической безопасности.

## 2. Текущий список

```text
/novostroyki/kvartiry-v-novostroykah/
/novostroyki/semejnaya-ipoteka/
/spravochnik/proverka-dokumentov-novostroyki/
/spravochnik/proektnaya-deklaratsiya/
/spravochnik/kak-vybrat-planirovku/
```

## 3. Правила статуса

Пока страница не проверена, в JSON должно быть:

```json
{
  "robots": "noindex,follow",
  "status": "draft"
}
```

После проверки возможны статусы:

```text
ready      — страница готова к финальному просмотру
published  — страницу можно добавить в sitemap и открыть к индексации
archived   — страница больше не нужна
```

## 4. Перед публикацией

Перед переводом SEO-страницы в `published` нужно проверить:

1. Нет ли неподтверждённых цен, условий ипотеки и обещаний наличия квартир.
2. Есть ли безопасный дисклеймер.
3. Не выглядит ли страница как официальный сайт застройщика.
4. Работают ли все ссылки.
5. Если есть форма, передаются ли `lead_type`, `form_id`, `project_id`, `project_name`.
6. Проходят ли проверки `npm run validate` и `npm run validate:json`.
