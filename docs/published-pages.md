## Что проверять после публикации

1. Открывается главная страница.
2. Открываются все URL из этой карты.
3. Формы показывают сообщение после отправки.
4. В исходном коде страниц есть корректные `title` и `description`.
5. `robots.txt` доступен по адресу `/robots.txt`.
6. `sitemap.xml` доступен по адресу `/sitemap.xml`.
7. Сайт добавлен в Яндекс.Вебмастер и Google Search Console.
