# Маршрутизация заявок

## Поля лида

| Поле | Назначение |
| --- | --- |
| `name` | Имя клиента |
| `phone` | Телефон |
| `interest` | Тип интереса |
| `payment` | Способ покупки, если есть в форме |
| `comment` | Комментарий клиента |
| `source` | URL страницы отправки |
| `page_title` | Заголовок страницы |
| `referrer` | Источник перехода |
| `tracking.utm_source` | Источник рекламы |
| `tracking.utm_medium` | Тип размещения |
| `tracking.utm_campaign` | Кампания |
| `tracking.realtor` | Идентификатор риэлтора |
| `tracking.realtor_id` | Внутренний ID риэлтора |
| `created_at` | Время создания заявки |

## Правила распределения

| Условие | Куда направлять |
| --- | --- |
| Есть `realtor` | Закрепить за этим риэлтором |
| `interest = Ипотека` | Специалист по ипотеке + ответственный риэлтор |
| `source` содержит `/investoram/` | Риэлтор с опытом инвестиционных сделок |
| `source` содержит `/semeyam/` | Риэлтор по семейным покупкам |
| `source` содержит `/pokupka-s-prodazhey/` | Риэлтор, который умеет вести цепочки сделок |
| `source` содержит `/dokumenty/` | Риэлтор + юрист/ответственный по документам |
| Нет `realtor` | Дежурный риэлтор или общий пул |

## Статусы лида

1. Новый.
2. Взят в работу.
3. Не дозвонились.
4. Уточнен запрос.
5. Ждёт планировки.
6. Ждёт цены.
7. Ипотечная консультация.
8. Нужно продать свою недвижимость.
9. Отложенный спрос.
10. Неактуально.

## Минимальный регламент звонка

1. Представиться и назвать источник заявки.
2. Уточнить, что заявка предварительная и не является бронью.
3. Уточнить тип квартиры.
4. Уточнить бюджет и способ покупки.
5. Узнать, нужна ли ипотека или продажа своей недвижимости.
6. Зафиксировать вопросы клиента.
7. Объяснить, когда и по какому поводу будет следующий контакт.

## Что считать качественной заявкой

Качественная заявка содержит:

- корректный телефон;
- понятный интерес;
- источник перехода;
- ответственного риэлтора или правило назначения;
- следующий шаг после первого контакта.
