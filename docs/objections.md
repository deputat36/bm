# Ответы на вопросы и возражения

## Назначение

Инструмент `/tools/objections/` нужен для единых ответов риэлторов на частые вопросы покупателей.

Он помогает:

- не обещать неподтверждённые цены;
- не заявлять официальный статус сайта;
- корректно объяснять предварительную заявку;
- фиксировать интерес клиента;
- переводить вопрос в следующий шаг.

## Адрес

```text
https://deputat36.github.io/bm/tools/objections/
```

Страница закрыта от индексации через `noindex, nofollow`.

## Основное правило

Если точных данных нет, нельзя придумывать ответ. Правильная схема:

1. Признать вопрос.
2. Объяснить, что данные пока не подтверждены.
3. Предложить зафиксировать интерес.
4. Обещать сообщить после появления подтверждённой информации.

## Частые темы

- цена;
- планировки;
- официальный статус сайта;
- бронирование;
- ипотека;
- сроки;
- документы;
- предчистовая отделка;
- своя котельная;
- зачем оставлять телефон.

## Что нужно согласовать

Перед активной рекламой желательно согласовать формулировки:

- с менеджером отдела продаж;
- с юристом;
- с ответственным застройщика или партнёром, если есть договорённость.
