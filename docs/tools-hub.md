# Хаб инструментов

## Назначение

Страница `/tools/` собирает внутренние инструменты отдела продаж по дому на Просторной 4А.

Она нужна, чтобы менеджеру или риэлтору не искать отдельные ссылки:

- генератор UTM-ссылок;
- ипотечный калькулятор;
- скрипт первого звонка;
- тест Web3Forms;
- боевая форма заявки;
- страница благодарности.

## Адрес

```text
https://deputat36.github.io/bm/tools/
```

Страница закрыта от индексации через `noindex, nofollow`.

## Кто использует

- менеджер отдела продаж;
- риэлторы;
- ипотечный специалист;
- человек, который запускает рекламу;
- человек, который проверяет Web3Forms.

## Порядок работы

1. Проверить Web3Forms через `/web3forms-test/`.
2. Создать ссылки риэлторов через `/tools/utm-builder/`.
3. При ипотечном вопросе открыть `/tools/mortgage-calculator/`.
4. После получения заявки использовать `/tools/call-script/`.
5. Фиксировать исход звонка по шаблону `data/call-outcomes-template.csv`.

## Что не делать

- Не отправлять клиентам ссылку на `/tools/`.
- Не публиковать `/tools/` в VK.
- Не использовать тестовую страницу Web3Forms в рекламе.
- Не хранить в инструментах персональные данные клиентов.
