# Генератор UTM-ссылок

## Назначение

Инструмент `/tools/utm-builder/` нужен, чтобы риэлторы и менеджеры быстро создавали персональные ссылки на сайт Просторной 4А.

По таким ссылкам в заявку попадают:

- `realtor`;
- `realtor_id`;
- `utm_source`;
- `utm_medium`;
- `utm_campaign`;
- `utm_content`.

## Адрес

После публикации GitHub Pages:

```text
https://deputat36.github.io/bm/tools/utm-builder/
```

Страница закрыта от индексации через `noindex, nofollow`.

## Как использовать

1. Выбрать страницу сайта.
2. Указать код риэлтора латиницей.
3. Указать источник трафика.
4. Указать тип размещения.
5. Указать кампанию.
6. Скопировать готовую ссылку.
7. Использовать ссылку в VK, Telegram, рекламе, сторис или QR-коде.

## Рекомендуемые коды кампаний

| Кампания | Когда использовать |
| --- | --- |
| `prostornaya_start` | общий запуск |
| `prostornaya_kvartiry` | реклама квартир |
| `prostornaya_mortgage` | ипотека |
| `prostornaya_family` | семейные покупатели |
| `prostornaya_invest` | инвесторы |
| `prostornaya_tradein` | покупка после продажи своей квартиры |
| `prostornaya_docs` | вопросы по документам |

## Правила

1. Один риэлтор - один постоянный код.
2. Не использовать ФИО кириллицей в `realtor`, лучше латиницу.
3. Для сторис чаще вести на `/contacts/`.
4. Для рекламы по ипотеке вести на `/ipoteka/`.
5. Для инвесторов вести на `/investoram/`.
6. Для семейных объявлений вести на `/semeyam/`.
