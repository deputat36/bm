# Стратегия микроразметки

Дата фиксации: 2026-07-07

## 1. Назначение

Микроразметка нужна для более понятной структуры страниц в поиске: хлебные крошки, карточка объекта, карточка сайта, карточка организации, статьи и FAQ.

На текущем этапе добавлена техническая заготовка:

```text
/assets/js/schema.js
```

Она может автоматически создавать JSON-LD для:

- `WebSite` — на основе data-атрибутов `body`;
- `BreadcrumbList` — на основе блока `.breadcrumbs`;
- `Residence` — на основе data-атрибутов `body`.

## 2. Подключение

Скрипт нужно подключать только после проверки страницы:

```html
<script src="/assets/js/schema.js"></script>
```

Для вложенных страниц путь должен соответствовать уровню вложенности:

```html
<script src="../../assets/js/schema.js"></script>
<script src="../../../assets/js/schema.js"></script>
<script src="../../../../assets/js/schema.js"></script>
```

## 3. WebSite

Для главной страницы портала можно добавить data-атрибуты на `body`:

```html
<body
  data-schema-website="Новостройки Борисоглебска"
  data-schema-website-url="https://tellermanovsad.ru/"
  data-schema-description="Городской портал по новостройкам Борисоглебска">
```

После этого скрипт сформирует объект `WebSite`.

Важно: `WebSite` должен описывать именно портал, а не официальный сайт застройщика или отдельного ЖК.

## 4. BreadcrumbList

Скрипт ищет блок:

```html
<div class="breadcrumbs"><a href="/">Главная</a> / <a href="/zhk/">ЖК</a> / Теллерманов сад</div>
```

И формирует:

```text
BreadcrumbList
```

Важно:

- текст хлебных крошек должен быть понятным;
- ссылки должны вести на реальные страницы;
- последняя часть без ссылки тоже попадёт в микроразметку.

## 5. Residence

Для карточки ЖК можно добавить data-атрибуты на `body`:

```html
<body
  data-schema-project="tellermanov-sad"
  data-schema-project-name="ЖК Теллерманов сад"
  data-schema-address="ул. Просторная, 4А"
  data-schema-city="Борисоглебск"
  data-schema-region="Воронежская область"
  data-schema-floors="9">
```

После этого скрипт сформирует объект `Residence`.

Важно: данные в микроразметке должны совпадать с видимым текстом на странице.

## 6. Уже подключено

На текущем этапе подключено:

```text
/zhk/tellermanov-sad/
```

На этой странице формируются:

- `BreadcrumbList`;
- `Residence`.

Для `/portal-preview/` подготовлена поддержка `WebSite`, но подключение нужно сделать после финальной визуальной проверки страницы и перед переносом в корневой `index.html`.

## 7. Что нужно добавить позже

- `Organization` для карточек застройщиков.
- `Article` для новостей и справочника.
- `FAQPage` для FAQ.
- `ItemList` для каталога новостроек.
- `Product` или `Offer` использовать осторожно: только если есть подтвержденные цены и юридически безопасные условия публикации.

## 8. Правила безопасности

1. Не добавлять `Offer` с ценой, если цена не подтверждена.
2. Не делать сайт похожим на официальный сайт застройщика.
3. Не указывать неподтвержденные данные как факт.
4. Микроразметка должна совпадать с видимым содержанием страницы.
5. Перед включением страницы в sitemap проверить JSON-LD валидатором.

## 9. Ближайшие шаги

1. Проверить JSON-LD на `/zhk/tellermanov-sad/` после публикации.
2. Подключить `WebSite` на будущей корневой главной после переноса `/portal-preview/`.
3. Добавить `Organization` для `/zastroyschiki/bm-group/`.
4. Добавить `ItemList` для `/novostroyki/` и `/zhk/`.
5. Добавить `Article` для объектной новости.
