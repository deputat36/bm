# Доменная стратегия для сайта ЖК «Теллерманов сад»

## Главная рекомендация

Основной домен для запуска:

**tellermanovsad.ru**

Почему именно он:

- совпадает с названием проекта без лишних слов;
- хорошо выглядит в рекламе, на баннерах, в VK и Telegram;
- легко диктовать по телефону;
- не привязан к конкретному адресу, если позже сайт расширится на весь ЖК;
- подходит под поисковые запросы: `Теллерманов сад`, `ЖК Теллерманов сад`, `Теллерманов сад Борисоглебск`.

## Что купить дополнительно, если свободно

### 1. zhk-tellermanov-sad.ru

SEO-вариант под запросы с приставкой ЖК.

Использовать как дополнительный домен с редиректом на основной.

### 2. tellermanov-sad.ru

Вариант с дефисом. Удобен для читаемости, но длиннее.

Можно купить, чтобы конкуренты не заняли похожий домен.

### 3. теллермановсад.рф

Кириллический домен для офлайн-рекламы и аудитории, которая плохо воспринимает латиницу.

Минусы: не всегда удобно отправлять ссылкой, возможны проблемы с копированием/кодировкой в некоторых сервисах.

### 4. zhkprostornaya4a.ru

Адресный домен под поисковые запросы `Просторная 4а Борисоглебск`, `новостройка Просторная 4а`.

Лучше использовать как вспомогательный редирект или отдельную SEO-посадочную, а не как основной бренд.

## Что не рекомендую как основной домен

### bm-borisoglebsk.ru

Минус: выглядит как сайт застройщика или официальная структура BM Group, что может создать юридическую и репутационную путаницу.

### etagi-tellermanov.ru

Минус: привязка к бренду «Этажи» в домене может потребовать согласования и может быть нежелательной для рекламной независимости.

### prostornaya4a.ru

Плюс: точное попадание по адресу.
Минус: слишком техническое название, слабее для бренда ЖК.

## Что проверить перед покупкой

1. Свободен ли домен.
2. Нет ли уже зарегистрированного сайта у застройщика с похожим доменом.
3. Нет ли ошибок в написании: правильно `Теллерманов`, с двумя буквами `л`.
4. Можно ли подключить домен к GitHub Pages.
5. Можно ли сделать SSL-сертификат через GitHub Pages.
