# Рекламные ссылки

## Назначение

Файл `data/ad-links-template.csv` содержит заготовки ссылок для VK-рекламы, постов и персональных ссылок риэлторов.

## Как использовать

1. Выбрать страницу под мотив объявления.
2. Собрать UTM-ссылку через `/tools/utm-builder/`.
3. Добавить `realtor` и `realtor_id`, если ссылку публикует конкретный специалист.
4. Проверить, что заявка сохраняет UTM и риэлтора.

## Основные связки

- Объявление про подбор квартиры -> `/podbor-kvartiry/`.
- Пост "ответим на вопросы по новостройке" -> `/konsultatsiya-po-novostroyke/`.
- Интрига и прогрев -> `/ranniy-dostup/`.
- Широкая реклама по новостройкам -> `/novostroyki-borisoglebsk/`.
- Запросы "от застройщика" -> `/kvartiry-ot-zastroyschika/`.

## Правило

Одна рекламная мысль - одна посадочная страница. Не нужно вести все объявления на главную, если можно дать человеку страницу под его конкретный интерес.
