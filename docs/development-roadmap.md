# План развития портала «Новостройки Борисоглебска»

Дата обновления: 2026-07-14

## Цель

Развивать независимый городской каталог новостроек Борисоглебска по пути:

```text
информация → проверка → сравнение → короткая заявка → консультация
```

Приоритет: Просторная 4А, Аэродромная 18Г, Сенная 76 и общий подбор.

Правило публикации:

```text
нет подтверждённого источника → нет публичного факта
```
