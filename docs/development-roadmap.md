# План развития портала «Новостройки Борисоглебска»

Дата обновления: 2026-07-15

## Цель

Развивать независимый городской каталог новостроек Борисоглебска по пути:

```text
информация → проверка → сравнение → короткая заявка → консультация
```

Приоритет: Просторная 4А, Аэродромная 18Г, Сенная 76 и общий подбор.

Правило публикации:

```text
нет подтверждённого источника → нет публичного факта
```

## Автономная зона

Без отдельного решения допускаются структура, навигация, неперсональная аналитика, guards, CI, документация и реестры.

Отдельного решения требуют реальная отправка, цены, снятие `noindex`, спорные изображения, юридические реквизиты, публикация рекламы, CRM, серверные редиректы и удаление legacy-страниц.
