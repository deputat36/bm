# План разработки портала «Новостройки Борисоглебска»

Дата фиксации: 2026-07-05

## 1. Рабочая логика

Проект развивается постепенно: сначала документация и архитектура, затем аккуратное расширение структуры, после этого перенос текущего сайта ЖК «Теллерманов сад» внутрь портала.

Главный принцип: не ломать текущий сайт и не терять заявки.

## 2. Этап 0 — аудит и фиксация текущего состояния

Статус: начато.

Что уже проверено:

- доступ к репозиторию `deputat36/bm` есть;
- права на push есть;
- основная ветка: `main`;
- проект сейчас является статическим сайтом одного объекта;
- текущая главная страница ориентирована на ЖК «Теллерманов сад»;
- текущая карта сайта использует домен `tellermanovsad.ru`;
- форма заявок реализована в `assets/js/main.js`.

Что нужно дополнительно проверить:

- полный список фактических папок и страниц;
- наличие всех изображений и рендеров;
- работу галереи;
- работу формы заявки после публикации;
- текущие настройки GitHub Pages и домена;
- наличие файла `CNAME`.
