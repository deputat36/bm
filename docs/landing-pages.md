Назначение: покупатели, которые рассматривают квартиру для аренды, перепродажи или сохранения капитала.

Запросы:

- квартира в новостройке Борисоглебск для инвестиций;
- купить квартиру для сдачи Борисоглебск;
- инвестиция в недвижимость Борисоглебск;
- квартира на Просторной 4А инвесторам.

Реклама:

```text
https://deputat36.github.io/bm/investoram/?utm_source=vk&utm_medium=ad&utm_campaign=prostornaya_invest
```

## Как использовать

1. Для платной рекламы вести трафик на страницу, совпадающую с мотивом объявления.
2. Для VK-постов использовать эти страницы как продолжение темы поста.
3. После подключения Метрики сделать отдельные цели/сегменты по этим URL.
4. После появления новых данных добавить ссылки на эти страницы из главной и раздела квартир.
5. Если нет возможности обновить основной `sitemap.xml`, отправить в поисковые панели `sitemap-extra.xml`.
