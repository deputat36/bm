# Чек-лист запуска рекламы

## Перед запуском

- [ ] Выбрана посадочная страница под конкретный мотив объявления.
- [ ] Текст собран или проверен через `/tools/ad-copy-builder/`.
- [ ] UTM-ссылка собрана через `/tools/utm-builder/`.
- [ ] Если ссылку публикует риэлтор, добавлены `realtor` и `realtor_id`.
- [ ] Web3Forms подключен и тестовая заявка дошла.
- [ ] На странице есть форма или явная ссылка на форму.
- [ ] Нет обещания брони, скидки, фиксированной цены или лучших квартир.
- [ ] Нет неподтверждённых цен, сроков, планировок и документов.
- [ ] Страница открывается на телефоне.
- [ ] Заявка попадает ответственному человеку.

## Что проверять после запуска

- [ ] Есть переходы по UTM.
- [ ] В заявке сохраняется `utm_campaign`.
- [ ] В заявке сохраняется `realtor`, если ссылка персональная.
- [ ] Есть звонки или сообщения по теме объявления.
- [ ] Некачественные заявки помечаются в отчёте.
- [ ] Лучшие связки заносятся в `data/ad-links-template.csv`.

## Минимальный тест

1. Открыть рекламную ссылку в режиме инкогнито.
2. Проверить, что страница соответствует тексту объявления.
3. Отправить тестовую заявку.
4. Проверить письмо Web3Forms.
5. Проверить, что UTM и страница заявки сохранились.

## Приоритетные страницы

- `/ranniy-dostup/` - прогрев и интрига.
- `/podbor-kvartiry/` - заявки на подбор.
- `/konsultatsiya-po-novostroyke/` - осторожная аудитория.
- `/novostroyki-borisoglebsk/` - широкий спрос.
- `/kvartiry-ot-zastroyschika/` - спрос по покупке в новом доме.
