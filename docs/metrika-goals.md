# Цели Яндекс.Метрики и аналитики

## Основные цели запуска

| Цель | Событие | Где использовать |
| --- | --- | --- |
| Клик по телефону | `phone_click` | Все ссылки на `8 903 857-69-09` |
| Отправка заявки | `lead_submit` | Все формы после подключения обработчика |
| Ошибка заявки | `lead_error` | Если форма не отправилась |
| Переход в квартиры | `apartments_open` | Ссылки на `/kvartiry/` |
| Переход в галерею | `gallery_open` | Ссылки на `/galereya/` |
| Переход на документы | `documents_open` | Ссылки на `/dokumenty/` |
| Переход на проектную декларацию | `declaration_open` | Ссылки на `/proektnaya-deklaratsiya-36-001139/` |
| Переход в ипотеку | `mortgage_open` | Ссылки на `/ipoteka/` |
| Переход на цены | `prices_open` | Ссылки на `/ceny/` |
| Переход по UTM риэлтора | сегмент `realtor` | Оценка работы каждого специалиста |

## Приоритет внедрения

1. `phone_click`
2. `lead_submit`
3. `lead_error`
4. `apartments_open`
5. `gallery_open`
6. `documents_open`
7. `declaration_open`
8. `mortgage_open`
9. `prices_open`

## Сегменты

Создать сегменты:

- посетители с `utm_campaign = tellermanov_start`;
- посетители с `utm_campaign = tellermanov_kvartiry`;
- посетители с `utm_campaign = tellermanov_mortgage`;
- посетители с `utm_campaign = tellermanov_documents`;
- посетители с параметром `realtor`;
- посетители страницы `/kvartiry/`;
- посетители страницы `/galereya/`;
- посетители страницы `/dokumenty/`;
- посетители страницы `/proektnaya-deklaratsiya-36-001139/`;
- посетители страницы `/ceny/`;
- посетители страницы `/ipoteka/`.

## Минимальные отчёты

1. Источники трафика.
2. Страницы входа.
3. Конверсии по формам.
4. Конверсии по кликам на телефон.
5. Конверсии по UTM-кампаниям.
6. Конверсии по риэлторам.
7. Переходы на квартиры, галерею и документы.
8. Отказы по посадочным страницам.

## Что считать хорошим результатом на раннем этапе

На этапе до старта продаж важны не только заявки, но и признаки спроса:

- переходы из VK и Telegram на сайт;
- клики по телефону;
- переходы в `/kvartiry/`;
- переходы в `/galereya/`;
- переходы в `/dokumenty/`;
- повторные визиты;
- заявки с указанным типом квартиры;
- вопросы по ценам, планировкам, ипотеке и документам.

## Важное ограничение

Пока Web3Forms, Supabase или CRM-обработчик не подключены, цель `lead_submit` нельзя считать полноценной заявкой. До подключения обработчика основными целями считаем `phone_click`, `apartments_open`, `gallery_open` и `documents_open`.
