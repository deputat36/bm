# Лимиты текстовых полей заявки

Дата обновления: 2026-07-15

## Назначение

Свободные текстовые поля ограничены, чтобы случайный или программный ввод не создавал чрезмерно большой delivery-пакет и не ухудшал обработку заявки.

Ограничения не подтверждают достоверность введённых сведений и не заменяют проверку оператором.

## Лимиты

| Поле | Максимум символов |
|---|---:|
| `name` | 80 |
| `budget` | 120 |
| `callback_time` | 120 |
| `convenient_time` | 120 |
| `comment` | 1000 |
| `question` | 1000 |

Телефон проверяется отдельным правилом: от 10 до 15 цифр и максимум 24 символа форматированного ввода.

## Два уровня защиты

### Интерфейс

Для найденных `input` и `textarea` устанавливается `maxlength`. Пользователь получает обычное браузерное ограничение ещё до отправки.

### Delivery

Полный объект заявки нормализуется дважды:

1. после `collectFormData`;
2. непосредственно перед `sendLead`.

Это сохраняет защиту при программной отправке или изменении HTML через DevTools.

## Нормализация

Для разрешённых свободных полей:

- удаляются непечатные управляющие символы;
- удаляются пробелы по краям;
- значение обрезается до установленного лимита.

Переносы строк в комментариях сохраняются.

## Что не изменяется

Нормализация не применяется к:

- телефону;
- рекламной атрибуции;
- согласиям;
- техническому ID обращения;
- объекту и форме;
- системным полям проекта.

Для этих данных действуют отдельные контракты и validators.

## Автоматическая проверка

Файл:

```text
tools/validate-lead-text-limits.mjs
```

Проверка подтверждает:

- точный список из шести полей;
- установленные значения лимитов;
- отсутствие контактов и служебных полей в списке;
- очистку управляющих символов;
- нормализацию после сбора данных;
- повторную нормализацию перед delivery;
- установку `maxlength` на элементы формы;
- сохранение device-privacy удаления `user_agent`.

## Ограничения

CI не заменяет ручную проверку вставки длинного текста в Chrome, Firefox, Safari, Android и iPhone. Фактический размер письма Web3Forms в этом изменении не проверяется.
