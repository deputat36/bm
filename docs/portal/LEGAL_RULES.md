# Юридически безопасные формулировки портала

Дата обновления: 2026-07-12

## Позиционирование

`novostroyki-borisoglebsk.ru` — независимый городской каталог новостроек Борисоглебска.

Портал не действует от имени конкретного застройщика и не должен создавать впечатление официального сайта жилого комплекса, застройщика или отдела продаж.

## Коммерческий приоритет

Отдельные формы заявок используются для трёх объектов:

- Просторная 4А;
- Аэродромная 18Г;
- Сенная 76.

Остальные объекты размещаются как справочные карточки и ведут на общую форму подбора.

## Запрещённые обещания

Нельзя утверждать без документального подтверждения:

- что портал является официальным представителем;
- что цена поступила напрямую от застройщика;
- что квартира забронирована;
- что стоимость зафиксирована;
- что конкретная квартира есть в наличии;
- что ипотека будет одобрена;
- что ставка или срок передачи ключей гарантированы.

## Безопасные формулировки

Использовать:

- «независимый городской каталог»;
- «предварительная заявка»;
- «получить консультацию»;
- «уточнить актуальную стоимость»;
- «наличие подтверждается перед консультацией»;
- «данные уточняются»;
- «информация проверяется по публичным источникам»;
- «решение по ипотеке принимает банк».

## Правила для форм

Рядом с формой должно быть понятно, что:

1. заявка не является бронью;
2. заявка не фиксирует цену;
3. наличие и условия подтверждаются отдельно;
4. пользователь соглашается на обработку персональных данных;
5. выбранный объект, источник перехода и рекламные метки фиксируются для обработки обращения.

## Правила для неподтверждённых объектов

Страницы Просторной 4А, Аэродромной 18Г и Сенной 76 остаются `noindex,follow`, пока данные не получат достаточное подтверждение.

Непроверенные сведения нельзя добавлять в sitemap, рекламу и SEO-тексты как установленные факты.

## Правила для справочного каталога

Справочная карточка публикуется только когда подтверждены:

- адрес;
- статус объекта;
- застройщик, если он указан;
- год ввода или срок сдачи, если он указан;
- хотя бы один публичный источник;
- дата последней проверки.

## Проверка перед публикацией

Перед открытием страницы для индексации нужно проверить:

- источники;
- дату актуальности;
- canonical;
- robots;
- форму заявки;
- дисклеймеры;
- отсутствие неподтверждённых обещаний;
- соответствие реестру объектов.
