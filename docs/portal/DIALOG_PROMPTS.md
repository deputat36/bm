# DIALOG PROMPTS для проекта «Новостройки Борисоглебска»

Дата обновления: 2026-07-09

Этот файл фиксирует промты для параллельных рабочих диалогов. Все диалоги работают на одну цель: превратить `novostroyki-borisoglebsk.ru` в городской портал-каталог всех новостроек Борисоглебска.

## Общее правило для всех диалогов

В конце каждого значимого ответа добавлять блок:

```text
Для управляющего диалога:
1. Что сделано.
2. Какие решения предлагаются.
3. Что нужно проверить юридически.
4. Что нужно передать в GitHub.
5. Что нужно добавить в базу объектов.
6. Что требует решения владельца.
```

Все финальные решения фиксируются в управляющем диалоге и в документах `docs/portal/`.

Задачи управляющего диалога:
1. Координировать работу всех направлений.
2. Проверять, не противоречат ли решения друг другу.

   docs/portal/CONTROL_CENTER.md

Формат каждого ответа:

4. Что нужно передать юридическому диалогу.
