# Фиксация интереса к справочным объектам

Дата обновления: 2026-07-12

Просторная 4А, Аэродромная 18Г и Сенная 76 остаются отдельными приоритетными объектами.

Для остальных новостроек используется справочный каталог и общая форма подбора. При нажатии на кнопку подбора портал сохраняет, с какой справочной карточки пришёл пользователь.

Передаваемые поля:

```text
residential_complex
residential_complex_id
reference_object_id
reference_object_name
reference_object_address
```

Для справочного объекта используется системный ID с префиксом `reference:`.

События аналитики:

```text
reference_catalog_action
mobile_lead_bar_click
```

Первое событие фиксирует переход к общей форме или открытие публичного источника. Второе фиксирует нажатие на мобильную кнопку заявки или звонка.

Справочная карточка выводится только после подтверждения источников и прохождения внутренней проверки. Нажатие на кнопку подбора не превращает объект в коммерчески приоритетный и не является обещанием цены, наличия или условий покупки.
