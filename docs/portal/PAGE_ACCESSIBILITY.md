# Доступность навигации по страницам

## Цель

Сделать основные публичные страницы портала удобнее для пользователей, которые работают с клавиатурой, экранным диктором или отключают анимации в настройках системы.

## Реализация

Добавлен общий runtime `assets/js/page-accessibility.js`.

Он:

- создаёт ссылку «Перейти к основному содержанию»;
- назначает основному блоку стабильный идентификатор `main-content`;
- делает `<main>` программно фокусируемым без добавления его в обычный Tab-порядок;
- переводит фокус к основному содержанию после активации skip-link;
- подключает общий файл стилей доступности;
- не читает и не сохраняет персональные данные;
- не изменяет формы, endpoint или аналитику.

## Подключение

На страницах с лид-формами runtime загружается из существующего `assets/js/conversion-tracking.js`.

Это охватывает в том числе:

- главную страницу;
- страницу консультации;
- каталог и другие действующие страницы с формами.

На странице `404.html`, где формы нет, runtime подключён напрямую.

## Видимый фокус

`assets/css/accessibility.css` задаёт контрастное синее focus-ring для:

- ссылок;
- кнопок;
- полей ввода;
- выпадающих списков;
- текстовых областей;
- элементов с разрешённым `tabindex`.

Фокус не маскируется и остаётся заметным на белом, красном и тёмном фоне.

## Уменьшение анимаций

При системной настройке `prefers-reduced-motion: reduce`:

- отключается плавная прокрутка;
- длительность анимаций и переходов сводится к минимуму;
- функциональность страниц сохраняется.

## Ограничения

Изменение не заменяет ручную проверку на реальных устройствах и с экранными дикторами. Оно закрывает только техническую базу клавиатурной навигации и reduced motion.

Количество лид-форм не изменяется:

- главная — 2;
- консультация — 2;
- 404 — 0.

## Автоматическая защита

Workflow `Page accessibility guard` проверяет:

- наличие runtime и CSS;
- skip-link и перевод фокуса;
- `focus-visible`;
- `prefers-reduced-motion`;
- подключение runtime на страницах с формами и на 404;
- неизменность количества форм;
- отсутствие формы на 404.
