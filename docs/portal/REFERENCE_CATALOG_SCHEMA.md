# Краткая карточка справочного объекта

Дата обновления: 2026-07-12

## Назначение

Просторная 4А, Аэродромная 18Г и Сенная 76 развиваются как отдельные лидогенерационные страницы.

Все остальные новостройки Борисоглебска добавляются сначала как краткие справочные карточки. Для них не создаётся отдельная продающая страница и отдельная форма, пока объект не получит подтверждённый коммерческий приоритет.

## Минимальный JSON

```json
{
  "id": "object-slug",
  "slug": "object-slug",
  "display_name": "Дом по адресу",
  "city": "Борисоглебск",
  "address": "Адрес",
  "builder_id": null,
  "builder_name": null,
  "building_status": "Данные уточняются",
  "commissioned_year": null,
  "handover": null,
  "floors": null,
  "wall_material": null,
  "commercial_role": "reference_catalog",
  "verification_status": "requires_check",
  "is_public_ready": false,
  "last_checked_at": "2026-07-12",
  "sources": []
}
```

## Обязательные поля

```text
id
slug
display_name
city
address
commercial_role
verification_status
is_public_ready
last_checked_at
sources
```

## Роли объектов

```text
priority_lead      отдельная карточка и отдельная форма заявки
reference_catalog  краткая справочная карточка без отдельной формы
```

## Что можно показывать

Для `reference_catalog` разрешено показывать только проверенные сведения:

1. адрес;
2. подтверждённый статус;
3. подтверждённого застройщика;
4. год ввода или срок сдачи;
5. этажность и материал дома;
6. ссылки на публичные источники;
7. дату последней проверки.

## Что нельзя показывать без актуального источника

```text
цены
наличие квартир
скидки
официальные планировки
условия бронирования
гарантированный срок передачи ключей
ипотечную ставку
утверждение о прямой продаже от застройщика
```

## Отображение в каталоге

Краткая карточка должна содержать:

```text
название или адрес
статус проверки
2–5 подтверждённых характеристик
дату обновления
ссылку на источники или пометку «источники проверяются»
```

Кнопки:

```text
Подробнее — только если есть подготовленная информационная страница
Подобрать новостройку — ведёт на общую форму портала
```

Кнопка «Оставить заявку по объекту» используется только для объектов с `commercial_role: priority_lead`.

## Правило индексации

```text
verification_status=confirmed + is_public_ready=true → можно рассматривать открытие индексации
requires_check или is_public_ready=false → noindex,follow и исключение из sitemap
```

## Следующий шаг

1. Собрать список новых многоквартирных домов Борисоглебска с 2018 года.
2. Для каждого адреса найти первичный публичный источник.
3. Добавить подтверждённые объекты в отдельный JSON-реестр.
4. Вывести их в справочной части `/catalog/`.
