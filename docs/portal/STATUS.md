## Справочный каталог

Справочные карточки выводятся только при выполнении всех условий:

```text
commercial_role=reference_catalog
verification_status=confirmed
is_public_ready=true
есть хотя бы один публичный источник
```

Пока подтверждённых карточек нет, пользователь видит безопасное сообщение о наполнении реестра. Приоритетные формы и общий подбор продолжают работать независимо от справочной части.

При выборе справочной карточки в заявку добавляются:

```text
reference_object_id
reference_object_name
reference_object_address
```

Проверка контролирует тип заявки, ID проекта, выбранный объект, системный ID объекта и наличие JavaScript обработки формы.

## Legacy-миграция

Первая очередь:

```text
/portal-preview/ → /
/novostroyki/ → /catalog/
/zhk/ → /catalog/
/zastroyschiki/ → /developers/
/spravochnik/ → /guides/
/novosti/ → /news/
```

Вторая очередь, уже обработано:

```text
/o-zhk/ → /catalog/prostornaya-4a/
/kvartiry/ → /catalog/prostornaya-4a/
/planirovki/ → /catalog/prostornaya-4a/
/ceny/ → /catalog/prostornaya-4a/
/dokumenty/ → /catalog/prostornaya-4a/
```

Для переходных страниц действуют правила:

```text
noindex,follow
нет старого домена tellermanovsad.ru
нет старого брендинга Этажи или BM Group
нет устаревших форм
есть явная ссылка на целевой раздел
нет meta refresh или JavaScript redirect
нет URL в sitemap.xml
```

Серверные редиректы ещё не включены. Все маршруты остаются `redirect_ready: false` до проверки целевых страниц и платформы размещения.

## Главные риски

1. В оставшихся глубоких legacy-страницах объекта могут быть canonical и брендинг прежнего проекта.
2. Аэродромная 18Г и Сенная 76 нельзя открывать в индекс до проверки данных.
3. Нельзя публиковать неподтверждённые цены, наличие, сроки и планировки.
4. Реальная доставка новых форм ещё не подтверждена контролируемым тестом.
5. Справочный каталог технически готов, но пока не наполнен подтверждёнными объектами.
6. Нужно проверить динамическую загрузку каталога и мобильную панель на опубликованном домене.
7. Платформа и формат серверных редиректов пока не зафиксированы.
8. Нужно решить, сохранять ли отдельными страницами галерею, инфраструктуру и ход строительства.
