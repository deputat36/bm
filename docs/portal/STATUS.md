# STATUS портала novostroyki-borisoglebsk.ru

Дата обновления: 2026-07-09

## Текущий статус

GitHub-доступ к `deputat36/bm` восстановлен. Управляющий контур проекта можно вести в репозитории.

Главная стратегия проекта подтверждена:

```text
novostroyki-borisoglebsk.ru = городской портал-каталог всех новостроек Борисоглебска
```

Главная страница должна быть каталогом объектов, а не страницей одного ЖК.

## Что уже есть

1. Домен `novostroyki-borisoglebsk.ru` начал работать.
2. В репозитории уже есть `docs/portal/CONTROL_CENTER.md`.
3. В проекте уже есть заготовки под портальную структуру.
4. Старый проект содержит много страниц, связанных с одним объектом — ЖК «Теллерманов сад» / Просторная 4А.
5. Есть отдельные рабочие диалоги по архитектуре, SEO, юридической безопасности и визуальному стилю.

## Главные риски

1. Корневой сайт может всё ещё восприниматься как сайт одного ЖК.
2. Часть старых URL, текстов, canonical и sitemap могут быть связаны с `tellermanovsad.ru`.
3. Нельзя публиковать неподтверждённые данные по ценам, наличию квартир, брони и условиям покупки.
4. Нельзя использовать формулировки, создающие впечатление официального сайта застройщика или ЖК.
5. Нельзя смешивать в одном разделе подтверждённые данные и предварительную рабочую информацию без маркировки.

## Не делать без отдельного решения

```text
- не удалять старые страницы;
- не менять массово canonical;

- не публиковать цены без подтверждения;
- не использовать «официальный сайт», «от застройщика», «официальный представитель»;
- не запускать рекламные страницы по брендовым запросам без юридической проверки.
```
