## Новые статьи

Активные версии расширены по сравнению со старыми черновиками:

- проверка документов включает объект, застройщика, разрешение, декларацию, землю, договор и фиксацию даты;
- материал о проектной декларации разделяет сведения о компании, участке, доме, квартирах, сроках и финансировании;
- материал о планировках учитывает полезную площадь, мебель, хранение, освещение, мокрые зоны и сценарии семьи.

Все три статьи сохраняют `noindex,follow` до редакционной и юридической проверки.

## Машиночитаемый контроль

Маршруты добавлены в:

```text
data/migration/legacy-routes.json
```

Параметры:

```text
redirect_phase=5
migration_action=retain_content
content_migration_status=migrated
redirect_ready=false
```

Проверка выполняется командой:

```bash
npm run validate:legacy
```

Серверные редиректы выпускаются только после определения платформы размещения и снятия блокирующих условий у целевых страниц.
