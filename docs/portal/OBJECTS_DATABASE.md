# База объектов портала novostroyki-borisoglebsk.ru

Дата фиксации: 2026-07-09

Репозиторий: `deputat36/bm`.

## 1. Назначение базы

Этот файл фиксирует рабочую базу новостроек Борисоглебска для будущего каталога.

Главная задача:

```text
не держать сайт вокруг одного ЖК, а собрать все новые дома и проекты города в единой структуре
```

База используется для:

1. каталога `/catalog/`;
2. карточек объектов `/catalog/{object}/`;
3. страниц застройщиков `/developers/`;
4. сравнений объектов;
5. SEO-страниц;
6. форм заявок;
7. будущего sitemap после публикации.

## 2. Принцип достоверности

Все объекты делятся по статусу проверки.

```text
draft                данные не подтверждены, можно использовать только как рабочий черновик
needs_verification   есть часть сведений, требуется проверка источников
partially_confirmed  часть данных подтверждена публичными документами или карточками
confirmed            данные сверены и можно готовить к публикации
published            страница опубликована и может быть в sitemap
```

Правило:

```text
если объект не подтверждён — is_public_ready: false и на странице noindex,follow
```

## 5. Приоритетные группы для исследования

### 5.1. Просторная 4А

Связанный объект: ЖК «Теллерманов сад».

Текущая задача:

1. сохранить старые страницы как legacy-контур;
2. развивать новую карточку `/catalog/prostornaya-4a/`;
3. не называть страницу официальным сайтом застройщика;
4. не публиковать цены без подтверждения;
5. вести заявки как интерес к объекту, а не как бронь.

### 5.2. Аэродромная 18Г

Текущая задача:

1. собрать сведения о доме;
2. определить застройщика;
3. проверить, связан ли объект с ПСК или другим застройщиком;
4. найти публичные документы;
5. заполнить карточку только подтверждёнными данными.

### 5.3. Сенная 76

Текущая задача:

1. собрать сведения о доме;
2. определить застройщика;
3. проверить статус: построен / строится / планируется;
4. найти документы и источники;
5. заполнить карточку только подтверждёнными данными.

### 5.4. Проекты ПСК

Текущая задача:

1. собрать список проектов ПСК в Борисоглебске;
2. отделить построенные дома от строящихся и планируемых;
3. определить, какие объекты можно показывать в каталоге;
4. подготовить страницу `/developers/psk/`;
5. не использовать чужой фирменный стиль и не создавать впечатление официального сайта ПСК.

### 5.5. Проекты BM Group

Текущая задача:

1. собрать все связанные объекты BM Group в Борисоглебске;
2. сохранить юридически безопасное позиционирование;
3. не использовать запрещённые формулировки;
4. подготовить страницу `/developers/bm-group/` как справочную страницу застройщика/группы проектов.

## 6. Объекты второй очереди

Сюда добавляются все новые дома Борисоглебска с 2018 года после проверки источников.

Минимальные поля для добавления:

```text
адрес
год ввода или срок сдачи
застройщик
статус объекта
этажность
тип дома
количество квартир, если есть источник
ссылка на источник
статус проверки
```

До подтверждения такие объекты нельзя открывать для индексации.

## 7. Связь с файлами данных

Текущие и планируемые файлы:


Новая портальная логика должна постепенно использовать объектный slug:

```text
prostornaya-4a
```
