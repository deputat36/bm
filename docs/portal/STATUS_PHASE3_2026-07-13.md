# Статус после третьей очереди legacy-аудита

Дата: 2026-07-13

## Итог

Портал работает как независимый городской каталог. Коммерческий приоритет сохраняют Просторная 4А, Аэродромная 18Г и Сенная 76.

В `data/migration/legacy-routes.json` зарегистрировано 19 старых маршрутов трёх очередей. Все связанные HTML-страницы переведены в режим `noindex,follow`, очищены от старого домена, прежнего брендинга, самостоятельных форм и автоматических перенаправлений.

## Третья очередь

```text
/galereya/                         retain_content
/infrastruktura/                   retire
/hod-stroitelstva/                 retire
/zastroyschik/                     redirect
/proektnaya-deklaratsiya-36-001139/ retain_content
/faq/                              retain_content
```

Для `retain_content` валидатор требует `content_migration_status`. Маршрут нельзя пометить готовым к редиректу, пока полезные данные не перенесены.

## Проверка

```bash
npm run validate:legacy
```

Проверяются исходные и целевые файлы, `noindex`, целевые ссылки, отсутствие старого домена и брендинга, отсутствие форм и автоматических перенаправлений, а также корректность `migration_action`.
