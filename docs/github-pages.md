# Публикация на GitHub Pages

Репозиторий: `deputat36/bm`.

После включения GitHub Pages сайт должен быть доступен по адресу:

```text
https://deputat36.github.io/bm/
```

## Настройка

1. Открыть `Settings -> Pages`.
2. В разделе `Build and deployment` выбрать `Source: GitHub Actions`.
3. Запустить workflow `Deploy GitHub Pages` вручную или сделать push в `main`.
4. Проверить сайт, `robots.txt` и `sitemap.xml`.

## После публикации

- Подключить Яндекс.Метрику.
- Добавить сайт в Яндекс.Вебмастер.
- Подключить обработчик заявок в `assets/js/main.js`.
- Добавить реальные планировки, цены, фото стройки и документы.
