# Операционный регламент сайта Просторная 4А

## Ежедневно

1. Проверить новые заявки.
2. Проверить, кто ответственный за каждую заявку.
3. Проверить заявки без статуса.
4. Проверить заявки без следующего шага.
5. Обновить список вопросов покупателей.

## После каждого рекламного поста

1. Проверить, что ссылка содержит UTM.
2. Проверить, что указан `realtor`, если пост публикует конкретный специалист.
3. Проверить переход по ссылке.
4. Проверить, что форма открывается.
5. Через 24 часа посмотреть заявки и источники.

## После появления новых данных

1. Обновить соответствующую страницу сайта.
2. Обновить статьи, где есть устаревшая формулировка.
3. Обновить `sitemap.xml`.
4. Добавить короткий пост во ВКонтакте.
5. Сообщить клиентам из списка раннего интереса.

## Каждую неделю

1. Посмотреть, какие страницы дают заявки.
2. Посмотреть, какие UTM-кампании дают заявки.
3. Посмотреть, какие риэлторы используют персональные ссылки.
4. Добавить одну новую SEO-статью или обновить существующую.
5. Проверить, нет ли заявок без обработки.

## Минимальная карточка заявки

- имя;
- телефон;
- интерес;
- источник;
- риэлтор;
- приоритет;
- следующий шаг;
- дата следующего контакта;
- комментарий.

## Главные риски

- заявки приходят, но не обрабатываются;
- риэлторы публикуют ссылки без UTM;
- клиенту обещают неподтверждённые цены или планировки;
- нет оператора персональных данных в юридических страницах;
- Web3Forms работает, но уведомления попадают в спам.
