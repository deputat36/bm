## Правила запуска

1. Не обещать скидки, дефицит квартир или бронирование без подтверждения.
2. Вести трафик на тематическую страницу, если объявление про ипотеку, цены или тип квартиры.
3. Для каждого риэлтора добавлять `realtor=slug`.
4. Для сторис вести на `/contacts/`.
5. После подключения Метрики создать цели на отправку формы и клики по телефону.
