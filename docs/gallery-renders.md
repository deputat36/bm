# Галерея рендеров проекта

На сайт добавлена отдельная страница:

`/galereya/`

Текущий статус:

- подключён JS-загрузчик `assets/js/gallery.js`;
- первые рендеры добавлены в текстовом формате base64, потому что текущий GitHub-коннектор блокирует безопасную массовую бинарную загрузку JPG через tree/commit;
- опубликована карта `sitemap-gallery.xml`;
- главную страницу нужно дополнительно связать с галереей через навигацию и блок “Рендеры проекта”, когда будет доступен обычный `git push` или точечное безопасное обновление `index.html`.

Что осталось сделать:

1. Загрузить все исходные JPG как обычные файлы:
   - `assets/img/gallery/zhk-prostornaya-4a-obshchiy-vid.jpg`
   - `assets/img/gallery/zhk-prostornaya-4a-fasad.jpg`
   - `assets/img/gallery/zhk-prostornaya-4a-vid-sverhu.jpg`
   - `assets/img/gallery/zhk-prostornaya-4a-fasad-front.jpg`
   - `assets/img/gallery/zhk-prostornaya-4a-plan-territorii.jpg`
   - `assets/img/gallery/zhk-prostornaya-4a-doma-i-dvor.jpg`
2. Переключить `gallery.js` с base64-файлов на обычные JPG.
3. Добавить ссылку “Галерея” в основное меню.
4. Добавить на главную блок с 2–3 самыми сильными рендерами.
