Инструменты отдела продаж

Закрытый от индексации набор рабочих страниц для запуска заявок, UTM-ссылок, ипотечных консультаций и первого звонка по дому на Просторной 4А.

Основные инструменты

Эти страницы не нужно продвигать в поиске. Их можно использовать внутри отдела продаж и в тестировании.

Генератор UTM-ссылок

Создаёт персональные ссылки риэлторов с `realtor`, `realtor_id` и UTM-метками.

Ипотечный калькулятор

Считает ориентировочный платёж, сумму кредита и комфортный доход.

Скрипт первого звонка

Помогает квалифицировать заявку и сформировать заметку для CRM или таблицы.

Тест Web3Forms

Проверяет `access_key` без сохранения ключа в репозитории.

Боевая форма

Основная страница отправки заявки после подключения Web3Forms.

Страница спасибо

Страница после успешной заявки, нужна для аналитики конверсии.

Порядок запуска

1

Проверить Web3Forms

Открыть тестовую страницу, вставить ключ, отправить тестовую заявку.

2

Собрать ссылки

Сформировать персональные ссылки риэлторов через UTM-генератор.

3

Обработать лид

После заявки использовать скрипт звонка и фиксировать следующий шаг.

Инструменты Просторная 4А
