# Следующие шаги по novostroyki-borisoglebsk.ru

## До покупки домена

- проверить доступность домена;
- купить `novostroyki-borisoglebsk.ru`;
- определить, будет ли сайт на GitHub Pages или обычном хостинге;
- создать отдельный репозиторий.

## После покупки домена

- перенести содержимое папки `novostroyki-borisoglebsk/` в новый репозиторий;
- подключить CNAME;
- настроить DNS;
- включить HTTPS;
- проверить открытие главной, каталога, контактов, ипотеки и гидов.

## Контент первой версии

- расширить главную страницу;
- сделать полноценный каталог;
- перенести раздел по первому проекту;
- добавить ипотечный раздел;
- добавить 3-5 инструкций покупателю;
- добавить нормальную политику обработки данных.

## Реклама

- запускать рекламу на общий каталог, а не на один объект;
- избегать брендовых запросов конкретных проектов без согласования;
- все объявления вести на нейтральные страницы подбора и каталога.
