# MVP novostroyki-borisoglebsk.ru

Рабочая заготовка будущего городского сайта по новостройкам Борисоглебска.

## Домен

`novostroyki-borisoglebsk.ru`

## Назначение

Не отдельный сайт одного ЖК, а городской каталог новостроек Борисоглебска.

## Структура

- `/` — главная городского сайта
- `/catalog/` — каталог новостроек
- `/catalog/tellermanov-sad/` — раздел ЖК «Теллерманов сад»
- `/ipoteka/` — ипотека и господдержка
- `/guides/` — инструкции покупателям
- `/contacts/` — консультация

## Принципы

- не имитировать официальный сайт застройщика;
- не использовать формулировки «официальный сайт», «квартиры от застройщика», «напрямую от застройщика»;
- продвигать общий каталог и услугу подбора;
- конкретные ЖК показывать как карточки внутри каталога;
- все заявки квалифицировать: имя, телефон, объект/формат, бюджет или способ покупки.
