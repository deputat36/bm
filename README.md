# Дом на Просторной 4А

Статический сайт для GitHub Pages: информационный сайт о новом доме на Просторной 4А в Борисоглебске.

## Назначение

- собрать предварительные заявки на квартиры;
- собирать заявки на ипотечные консультации;
- занять поисковые запросы по адресу, дому, планировкам, ценам и застройщику;
- регулярно добавлять новую информацию по проекту.

## Опубликованный адрес

После включения GitHub Pages сайт будет доступен по адресу:

```text
https://deputat36.github.io/bm/
```

## Важно

Сайт позиционируется как информационный ресурс агентства недвижимости. Он не является официальным сайтом застройщика.

## Заявки

В файле `assets/js/main.js` нужно заполнить `LEAD_ENDPOINT`, когда будет готов обработчик заявок: Supabase Edge Function, CRM webhook, Google Forms или другой endpoint.

Форма сохраняет UTM-метки и код риэлтора из URL:

```text
?realtor=ivanova&utm_source=vk&utm_medium=post&utm_campaign=prostornaya_start
```
